# E-Commerce Customer Analytics

A data analytics project covering 20 customers across 6 global markets (Jan–Mar 2024).

## Files
| File | Description |
|------|-------------|
| `customer_ecommerce_cleaned_dataset.xlsx` | Cleaned dataset with 20 records, 14 features |
| `ecommerce_analytics_report.pdf` | Visual report with charts and insights |

## Key Insights
- Total Revenue: $8,431
- Churn Rate: 15%
- Top Category: Electronics (68% of revenue)
- Avg Satisfaction: 4.0 / 5

## Data Cleaning Steps
1. Removed duplicates
2. Standardised date format (ISO 8601)
3. Handled missing values
4. Corrected data types
5. Engineered new features (Revenue, Segment, Churn flag)
